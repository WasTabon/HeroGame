# Draw Hero — Iteration 1: Foundation + Entry

## 1. Что было добавлено
Базовый каркас проекта: менеджеры (Sound, Haptic, Transition), главное меню, карта из 20 уровней со скроллом и переход на игровую сцену. Игровая сцена пока заглушка.

## 2. Что изменилось с прошлой итерации
Первая итерация — базы нет.

## 3. Установка

### Предварительно (обязательно)
1. Unity 2022.3.62f, Built-in 2D, платформа iOS, ориентация Portrait.
2. Установи **DOTween (free)** из Asset Store и нажми `Tools > Demigiant > DOTween Utility Panel > Setup DOTween`.
3. Проект использует **TextMeshPro** — при первом обращении Unity предложит импортировать TMP Essentials, согласись (`Window > TextMeshPro > Import TMP Essential Resources`).

### Файлы
Распакуй архив в корень проекта так, чтобы папка `Assets/DrawHero/` легла в `Assets/`. Дождись компиляции.

### Editor скрипт
В меню Unity запусти:
`Tools > DrawHero > Setup All Scenes (Iteration 1)`

Это создаст 4 сцены в `Assets/DrawHero/Scenes/` (Boot, MainMenu, LevelMap, Game), соберёт весь UI, прокинет ссылки и добавит сцены в Build Settings. После выполнения откроется сцена Boot.

## 4. Как тестировать
1. Открой сцену `Assets/DrawHero/Scenes/Boot.unity`.
2. Нажми Play.
3. Boot → авто-переход в MainMenu → жми PLAY → карта уровней → тапни любой уровень → игровая сцена-заглушка → BACK возвращает на карту.

## 5. Ожидаемый результат
- Заголовок DRAW HERO появляется с scale-punch (Ease.OutBack).
- Кнопки нажимаются с анимацией сжатия и звуком (сгенерированные тоны).
- Между сценами плавный fade-переход.
- Карта: 20 кнопок-уровней в сетке 3 колонки, скроллятся вертикально, появляются каскадом.
- Safe area учитывается (проверь на iPhone-разрешении с вырезом).

## 6. Известные ограничения
- Менеджеры (Sound/Haptic/Transition) создаются автоматически при старте игры через ManagerBootstrap — можно запускать Play с ЛЮБОЙ сцены, они появятся сами. В Boot-сцене их больше нет вручную (чтобы не было дубликатов).
- Все уровни на карте доступны (нет блокировки/прогрессии — будет в Итерации 4).
- Игровая сцена пустая (геймплей в Итерации 2).
- Звёзд на кнопках уровней ещё нет.
- Haptic вызывает Handheld.Vibrate (простая вибрация; taptic-градация — позже при желании).
